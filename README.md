# Motivational Quotes Automation with n8n

## Overview

This project is an automated workflow built with n8n.

The workflow runs every day at 10:00 AM and performs the following tasks:

- Retrieves a motivational quote from an external API.
- Translates the quote into Portuguese.
- Sends the translated quote automatically via email.

## Technologies

- n8n
- HTTP Request
- Translation API
- Gmail
- REST API

## Workflow

1. Schedule Trigger
2. Get Motivational Quote
3. Translate Quote
4. Send Email

## Purpose

This project was created to practice workflow automation, API integration, and AI-powered automation using n8n.
